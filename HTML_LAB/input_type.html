INPUT TYPE



Tag Description
button ใช้สร้างปุ่ม
checkbox ใช้สร้างตัวเลือกได้มากกว่า 1 ค่า หรือไม่เลือกเลย
color ใช้สร้างช่องกรอกสี
date ใช้สร้างช่องกรอกวันที่
datetime-local ใช้สร้างช่องกรอกวันที่และเวลา (ไม่มี time zone)
email ใช้สร้างช่องกรอกอีเมลโดยเฉพาะ
file ใช้เพื่อเป็นการ import file จากในเครื่อง
hidden ใช้ส่งข้อมูล ที่ไม่ต้องการให้ผู้เยี่ยมชมเว็บไซต์เห็นข้อมูลดังกล่าว
image ใช้สร้างรูปภาพให้กับฟอร์ม เพื่อส่งข้อมูลไปยัง form-handler
month ใช้สร้างช่องกรอกเดือนและปี
number ใช้สร้างช่องกรอกตัวเลข
password ใช้สร้างกล่องใส่รหัสผ่าน
radio ใช้สร้างตัวเลือกค่าใดค่าหนึ่ง
range ใช้สร้างช่องกรอกตัวเลขในระยะที่กำหนด
reset ใช้สร้างปุ่มรีเซ็ตให้กับฟอร์ม เพื่อรีเซ็ตค่าทุกช่องในฟอร์มกลับไปสู่ค่าเริ่มต้น
search ใช้สร้างช่องค้นหาโดยเฉพาะ
submit ใช้สร้างปุ่ม Submit ให้กับฟอร์ม เพื่อส่งข้อมูลไปยัง form-handler
tel ใช้สร้างช่องกรอกเบอร์โทรศัพท์
text ใช้สร้างกล่องใส่ข้อความบรรทัดเดียว
time ใช้สร้างช่องกรอกเวลา (ไม่มี time zone)
url ใช้สร้างช่องกรอก URL
week ใช้สร้างช่องกรอกสัปดาห์และปี
